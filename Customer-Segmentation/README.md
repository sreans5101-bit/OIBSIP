# Customer Segmentation Project

## Project Overview

Customer segmentation is the process of dividing customers into groups based on similar characteristics or purchasing behavior.

This project uses data analysis techniques to identify different types of customers.

## Objectives

* Understand customer purchasing behavior
* Group customers with similar characteristics
* Identify potential target groups for marketing

## Techniques Used

* Data preprocessing
* Data visualization
* Customer behavior analysis

## Tools and Libraries

* Python
* Pandas
* Matplotlib
* Seaborn
* Scikit-learn

## Files Included

* `customer_segmentation.ipynb` – Notebook containing segmentation analysis
* `customer_segmentation.pdf` – Project documentation

## Outcome

Customer segmentation helps businesses design targeted marketing strategies and improve customer satisfaction.
