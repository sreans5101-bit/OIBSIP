# Exploratory Data Analysis on Retail Sales

## Project Overview

This project performs Exploratory Data Analysis (EDA) on a retail sales dataset to discover patterns, trends, and customer behavior.

Exploratory Data Analysis helps summarize the main characteristics of the dataset using statistical techniques and visualizations.

## Dataset Features

The dataset contains information such as:

* Transaction ID
* Date
* Customer ID
* Gender
* Age
* Product Category
* Quantity
* Price per Unit
* Total Amount

## Project Objectives

* Understand the structure of the dataset
* Identify missing values and inconsistencies
* Analyze customer demographics
* Analyze product category performance
* Visualize sales trends

## Tools and Libraries

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

## Files Included

* `EDA_retail.ipynb` – Jupyter notebook containing analysis
* `EDA_on_retail.pdf` – Project documentation
* `retail_sales_dataset.csv` – Dataset used for analysis

## Outcome

The analysis helps identify sales patterns, customer purchasing behavior, and category performance which can assist businesses in making better marketing and sales decisions.
