# Data Cleaning Project

## Project Overview

This project focuses on preparing raw data for analysis by cleaning and preprocessing the dataset.

Data cleaning improves the quality of the dataset and ensures that it is suitable for analysis and machine learning tasks.

## Tasks Performed

* Handling missing values
* Removing duplicate records
* Correcting inconsistent data
* Data type conversion
* Preparing dataset for analysis

## Tools and Libraries

* Python
* Pandas
* NumPy

## Files Included

* `data_cleaning.ipynb` – Jupyter notebook containing cleaning steps
* `data_cleaning.pdf` – Project report

## Outcome

After cleaning, the dataset becomes more reliable and suitable for further data analysis and machine learning applications.
